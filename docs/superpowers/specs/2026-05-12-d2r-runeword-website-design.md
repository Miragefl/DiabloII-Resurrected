# D2R Runeword & Equipment Website - Design Spec

## Overview

A bilingual (Chinese/English) static website for Diablo II Resurrected players to browse and search all runewords, runes, and base items. Dark gothic visual style matching the game's aesthetic.

## Tech Stack

- **Framework**: Nuxt 3 (Vue 3, SSG mode)
- **Styling**: Tailwind CSS with custom dark gothic theme
- **i18n**: @nuxtjs/i18n (zh/en)
- **Data**: JSON files generated by scraper scripts
- **Deployment**: Vercel (auto-deploy from Git)

## Data Models

### Runeword

```typescript
interface Runeword {
  slug: string
  name: { en: string; zh: string }
  runes: string[]
  runeNumbers: number[]
  sockets: number
  itemTypes: string[]
  level: number
  ladderOnly: boolean
  d2rOnly: boolean
  patch: string
  effects: { en: string[]; zh: string[] }
  bestFor?: string[]
}
```

### Rune

```typescript
interface Rune {
  id: number
  name: string
  nameZh: string
  level: number
  weaponEffect: string
  armorEffect: string
  shieldEffect: string
  dropLevel: { normal: number; nightmare: number; hell: number }
}
```

### BaseItem

```typescript
interface BaseItem {
  slug: string
  name: { en: string; zh: string }
  type: string
  maxSockets: number
  level: number
  strength?: number
  dexterity?: number
  damage?: { min: number; max: number }
  speed?: number
  defense?: { min: number; max: number }
  compatibleRunewords: string[]
}
```

## Pages & Features

### Home (/)
- Dark gothic hero with particle/dark elemental background
- Quick access links: Runewords, Runes, Base Items
- Featured/popular runewords showcase (Enigma, Infinity, etc.)
- Language toggle (top-right)

### Runewords List (/runewords)
- Card grid layout for all 99 runewords
- Filter panel (sidebar on desktop, bottom drawer on mobile):
  - Socket count (2-6)
  - Item type (Weapons/Armor/Shields/Helms etc.)
  - Level range (slider)
  - Ladder-only toggle
  - Recommended class
- Real-time search (name/effect keywords)
- Sort by: name/level/sockets
- Card content: name, rune sequence (with icons), sockets, level, summary effects

### Runeword Detail (/runewords/[slug])
- Large title + rune arrangement (icons + names with visual connection lines)
- Full effect list
- Compatible base items (linked)
- Crafting tips (base material requirements, rune drop sources)
- Prev/next navigation

### Runes List (/runes)
- Grid display of 33 runes
- Each shows: number, icon, name, level
- Clickable to detail

### Rune Detail (/runes/[id])
- Large rune icon + number + name
- Socket effects (weapon/armor/shield)
- Drop information
- All runewords using this rune

### Base Items List (/base-items)
- Grouped by type
- Filter by type/sockets/level
- Each shows: name, type, stats, max sockets

### Base Item Detail (/base-items/[slug])
- Full stats
- Compatible runewords list

### Responsive Design
- Mobile/tablet/desktop adaptive
- Mobile: filters collapse to bottom drawer
- Tablet: 2-column card grid
- Desktop: 3-4 column card grid + sidebar filters

## Visual Theme

### Colors
```
Background:   #0a0a0f (abyss black) / #12121a (dark purple-black)
Card bg:      #1a1a2e (dark indigo)
Primary:      #8b5cf6 (dark purple) - buttons/links/highlights
Accent:       #d4a017 (dark gold) - titles/rune icons/rare badges
Danger:       #dc2626 (dark red) - ladder-only/warnings
Text:         #e2e8f0 (light gray-white) / #94a3b8 (secondary gray)
Border:       #2d2d44 (dark purple border)
```

### Typography
- Headings: Cinzel (gothic serif)
- Body: Inter (clean readability)
- Chinese: Noto Sans SC

### Visual Elements
- Rune icons: SVG with dark gold glow effect
- Runeword cards: stone/metal texture borders, purple glow on hover
- Background: subtle dark texture + CSS gradients for depth
- Dividers: gothic geometric patterns

### Animations
- Page transitions: fade in/out
- Card hover: slight elevation + halo
- Filter operations: smooth transitions
- Keep it dark and沉稳, no flashy animations

## Data Collection Strategy

### Primary Source
- **diablo2.io** — most complete, updated to v3.1

### Backup Sources
- Wowhead D2R section
- 网易暗黑百科 (Chinese translations)
- d2r.world (Chinese data)

### Scraping Approach
1. Separate Node.js scripts for each data type (runewords, runes, base items)
2. Output standardized JSON to `data/` directory
3. Chinese translations scraped from Chinese wiki sources
4. Scripts can be re-run when new patches release

### Data Validation
- Schema validation after scraping (zod or similar)
- Cross-reference multiple sources for accuracy
- Manual review for critical data points

## SEO & Performance

- SSG pre-renders all pages for instant loading
- Semantic HTML structure
- Meta tags per page (title, description, og:image)
- Sitemap auto-generated by Nuxt
- Image optimization via Nuxt Image
